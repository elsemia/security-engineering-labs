# security-engineering-labs
laboratório de testes semanais focado em cybersegurança IOT integrado à ML.

Este repositório documenta meus estudos práticos em **engenharia de segurança**, com foco em:
- redes e análise de tráfego;
- segurança defensiva (IDS, detecção de anomalias);
- machine learning aplicado à segurança;
- sistemas e dispositivos IoT.

O objetivo é a construção de um entendimento técnico progressivo, partindo de fundamentos e evoluindo para sistemas reais.


## Estrutura do repositório

O repositório é organizado por semanas de estudo:
week-01/
week-02/
week-03/
...

Cada diretório semanal contém:
- experimentos práticos;
- scripts funcionais;
- anotações técnicas;
- um README descrevendo a pergunta estudada, a abordagem e os resultados.

---

## Metodologia

Para cada semana, sigo o mesmo processo:

1. Defino uma pergunta técnica concreta  
2. Implemento um ou mais experimentos para respondê-la  
3. Analiso os resultados (o que funcionou e o que não funcionou)  
4. Registro próximos passos ou limitações

---

## Tecnologias e ferramentas

As ferramentas variam ao longo do tempo, mas incluem:
- Linux

- Snort / Suricata

- ESP32 / Arduino
